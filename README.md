# Bit-OS

* **Status do Projeto:** Desenvolvimento Alpha (Não recomendado para uso sério ou produção)
* **Base:** Gentoo Linux
* **Init System:** OpenRC (Único suportado)
* **Segurança:** SELinux ativado por padrão / Segurança máxima
* **Arquitetura de Arquivos:** FHS Compliant com `usr-merged`
* **Bootloader:** GRUB (Único suportado)
* **Idioma:** Apenas Inglês (Suporte total por agora)
* **Plataformas:** `amd64` e Raspberry Pi 5

---

## Descrição

O **Bit-OS** é uma distribuição de propósito geral focada em segurança máxima. O sistema foi concebido para atender a diversos cenários de uso, incluindo:

* Jogar vídeo jogos
* Empresas e estações de trabalho
* Estudos e escolas
* Uso desktop
* Uso minimalista
* Modo Terminal (CLI)

---

## Edições e Ambientes de Trabalho

Por padrão, o Bit-OS instala um sistema puramente focado em **Modo Terminal (CLI)**. No entanto, oferece suporte oficial para a instalação dos seguintes ambientes gráficos:

* **KDE Plasma:** Versão padrão (*vanilla*) ou a versão modificada customizada para o Bit-OS.
* **GNOME:** Versão padrão (*vanilla*) ou a versão modificada customizada para o Bit-OS.

---

## Gerenciamento de Pacotes

O gerenciador de pacotes oficial e nativo da distribuição é o **bpm (Bit-OS Package Manager)**.

---

## Requisitos de Hardware

* **Memória RAM:** 2GB (Mínimo recomendado)
* **Armazenamento:** 40GB de espaço mínimo
* **Tipo de Disco:** SSD
